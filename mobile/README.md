# Frontend — Dental Clinic Admin Panel

React + TypeScript + Vite admin panel for the Dental Clinic Management System.

## Status

Frontend scaffold coming in Step 18. This folder will contain:

- React 18 + TypeScript + Vite
- Material UI (MUI)
- TanStack Query (API state)
- Zustand (local state)
- React Hook Form + Zod (forms + validation)
- React Router v6

## Roles

- **Admin** — full access, user management, reports
- **Receptionist** — appointments, patients, billing
- **Dentist** — own schedule, patient treatment notes

## Pages (planned)

- Login
- Dashboard (Receptionist / Dentist)
- Patients (list, create, profile)
- Appointments (list, calendar, create)
- Treatment Records
- Billing (invoices, payments)
- Reports
- User Management (Admin only)
- Audit Logs (Admin only)
